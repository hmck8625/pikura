## アップロード手順（手動）

### YouTube Studio からのアップロード

1. [YouTube Studio](https://studio.youtube.com) にログイン
2. 右上の「作成」→「動画をアップロード」
3. MP4ファイルをドラッグ&ドロップ
4. 上記のタイトル・説明文・タグをコピペ
5. 「Shorts動画としてサンプリングを許可」をON
6. 公開設定 → 「公開」or 「スケジュール設定」

### チャンネル概要欄のテンプレート

```
ピクラ（pikura）| 日本最大のピックルボール総合プラットフォーム

🏓 ルール解説・ギア比較・コート情報を発信中
📱 https://pikura.app
🏆 JPA公式ランキング124選手掲載中

#ピックルボール #pickleball
```
