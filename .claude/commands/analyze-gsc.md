# Google Search Console 分析

GSCからデータを取得し、SEO状況を分析してください。

## 手順

1. レポート生成: `node scripts/gsc-report.mjs`（直近28日間）
   - 期間指定: `node scripts/gsc-report.mjs --days 7`（直近7日間）
2. 出力されたMarkdownレポートを読み込み、以下を分析:
   - 検索クエリの傾向（どのキーワードで表示されているか）
   - ページ別のパフォーマンス（どのページが強いか）
   - CTR改善の余地（表示はあるがクリックされていないページ）
   - 順位が惜しいキーワード（11-20位 → 改善すれば1ページ目に入る）
   - デバイス分布（モバイル vs PC）
3. 改善提案を出す:
   - タイトル・description改善案
   - コンテンツ追加すべきキーワード
   - 内部リンク強化すべきページ

## 初回認証（トークンがない場合）
```
node scripts/analytics-auth.mjs
```

## 注意
- GSCのデータは2-3日遅れで反映されます
- サイト立ち上げ直後はデータが少ない/ゼロの場合があります
- レポートは `reports/gsc-report-{date}.md` に保存されます
