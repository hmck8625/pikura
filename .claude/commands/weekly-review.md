# 週次レビュー

pikura.app プロジェクトの週次レビューとAIチームMTGを実施します。

## 手順

1. 現在のプロジェクト状態を確認する
   - `git log --oneline -20` で直近のコミット履歴
   - Search Console / GA のデータ（ユーザーから共有があれば）
   - 記事数、ページ数の確認
2. `/mnt/c/Users/daisu/Desktop/ピ/AIチーム定義.md` を読み込み、チームメンバーのペルソナで議論
3. 以下の観点でレビュー:
   - 前週のアクションアイテム達成状況
   - KPI進捗（PV、フォロワー数、記事数、検索インプレッション）
   - 技術的な課題・バグ
   - SEOの状態（インデックス数、検索順位）
   - SNS運用の振り返り
   - 次週のアクションプラン
4. MTGドキュメントを `/mnt/c/Users/daisu/Desktop/ピ/week{N}/` に保存
5. 中村PM（リアリスト）の視点で現実チェックを必ず入れる

## レビュー対象スキルの確認

各スキル定義ファイル（`.claude/commands/*.md`）の内容が最新かも確認し、必要に応じて更新する。

### 定義済みスキル（6つ）

| # | スキル | 用途 |
|---|--------|------|
| 1 | `/generate-tweets` | X投稿候補の自動生成 |
| 2 | `/generate-eyecatch` | 記事アイキャッチ画像の生成 |
| 3 | `/generate-video-script` | ショート動画台本の生成 |
| 4 | `/generate-short` | 記事→ショート動画の一括生成パイプライン |
| 5 | `/import-articles` | microCMSへの記事一括入稿 |
| 6 | `/weekly-review` | 週次レビュー・AIチームMTG |
